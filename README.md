# docker web dev base
docker-composeを用いて、nginx/PHP/mysqlな開発環境を一発構築するための最低限が揃ったrepository

# install
次のコマンドを実行

`docker-compose build`

`docker-compose up -d`

# use
下記にアクセス

## nginx/public_html
http://localhost:18080/

## phpmyadmin
http://localhost:8888/

ID: root
PW: secret
